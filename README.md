# Caaaars - A sample app to test react, redux and following tech.
## Purpose
Trying out react router and teaching self.

## Libs
This app uses the following
 - React
 - redux
 - react-hot-loading
 - Webpack
 - react-bootstrap
 - react-router
 - eslint-airbnb
 - promise-middleware
 - redux-thunk
 - karma
 - mocha
 - phantom
 - bootstrap 4 alpha

## Setting up

```
npm install
npm install -g eslint
```

## Starting the server

```
npm start
```

## Running tests
```
karma start
```
